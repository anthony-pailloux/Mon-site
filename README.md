# Mon Site — Exercice HTML & CSS pour débutants

Projet simple pour apprendre à créer sa première page web.

## Fichiers du projet

```
Mon-site/
├── index.html    ← le contenu (structure)
├── style.css     ← l'apparence (couleurs, espacements)
└── README.md     ← ce fichier
```

## Comment voir le site

Ouvre `index.html` dans ton navigateur (double-clic).

## Ce que tu apprends

### HTML
- Les balises de base : `html`, `head`, `body`
- Les titres : `h1`, `h2`
- Le texte : `p`, `strong`, `em`
- Les listes : `ul`, `ol`, `li`
- Les liens : `a`
- Les images : `img`
- Le formulaire : `form`, `label`, `input`, `textarea`, `button`
- Les sections : `header`, `nav`, `main`, `section`, `footer`

### CSS
- Changer la couleur : `color`, `background-color`
- Les espacements : `margin`, `padding`
- Les bordures : `border`
- Centrer du texte : `text-align`
- Styliser un bouton : `.bouton`
- Effet au survol : `:hover`

## Exercices pour progresser

1. Change ton prénom dans le texte
2. Change la couleur bleue `#4a90d9` par une autre couleur
3. Ajoute une nouvelle section "Mes projets"
4. Remplace l'image par ta propre photo
